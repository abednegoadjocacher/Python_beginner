DJANGO INSTALLATION

What is Django? Django is a web framework for python. To install Django on your PC; First you need to have python install on your PC. Incase you do not have python installed on your PC, click on the link to download python. Download here. Now that you have installed python successfully, let's continue to installing Django.

  • Open command prompt on your PC.
  • Type the following and press the Enter key to see the version of python install: python --version
    • Type the next line to install Django: pip install django
This command install django successfully on your PC
